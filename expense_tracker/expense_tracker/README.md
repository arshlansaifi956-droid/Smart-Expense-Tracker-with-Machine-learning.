## 🔍 Overview

Smart Expense Tracker is a feature-rich web application that combines traditional expense management with modern AI capabilities. It allows individuals and families to log expenses, set budgets, scan receipts for automatic data extraction, and receive intelligent financial insights — all through an intuitive, responsive dashboard.

---

## ✨ Features

### Core Expense Management
- **Expense Logging** — Add, edit, and delete expenses with custom categories and icons
- **Budget Management** — Set monthly budgets with real-time usage tracking and visual indicators
- **Expense Calendar** — Visualize spending patterns on an interactive monthly calendar
- **CSV & PDF Export** — Download expense reports in CSV or PDF format via ReportLab

### AI & Machine Learning
- **Spending Forecast** — Random Forest Regressor predicts total spending for the next 7 days based on historical patterns (day-of-week, day-of-month, weekend effects)
- **Savings Recommendations** — Automated tips generated by comparing current vs. previous 30-day spending per category, with priority levels (Low / Medium / High / Critical)
- **Budget Alerts** — Automatic notifications at 80% and 100% budget utilization

### Natural Language & Voice Input
- **NLP Expense Parsing** — Type expenses in plain English or Hindi (e.g., *"Yesterday I spent 300 on groceries"*) and have them auto-categorized using a zero-shot DistilBERT classifier
- **spaCy Date Extraction** — Named entity recognition (NER) for natural date parsing
- **Hindi Keyword Mapping** — Built-in vocabulary bridge for common Hindi expense terms (किराना, खाना, किराया, etc.)
- **Voice Assistant** — Speech-to-text input supporting English (`en-US`) and Hindi (`hi-IN`) via the Web Speech API

### Receipt OCR
- **PaddleOCR Integration** — Upload receipt images for automatic text extraction and expense parsing
- **Multi-engine Fallback** — Falls back to Tesseract OCR if PaddleOCR is unavailable
- **Receipt Gallery** — Browse and manage all uploaded receipts

### Group & Family Expenses
- **Family Groups** — Create groups, add members, and log shared expenses
- **Group Expense Splitting** — Track who paid and manage group balances

### Reports & Analytics
- **Dashboard Analytics** — Charts showing spending by category, trends over time, and budget progress
- **Search & Filter** — Full-text search across expense names and categories
- **Reports Module** — Monthly summaries with visual breakdowns

### Personalization
- **Dark / Light Mode** — Toggle between themes, saved per user
- **Multi-currency Support** — Configure currency and symbol (default: INR ₹)
- **Notification Preferences** — Enable or disable in-app alerts
- **Multi-language Voice** — Enable multiple languages for the voice assistant

## 🚀 Getting Started

### Prerequisites

- Python 3.10 or higher
- `pip` package manager
- (Optional) A virtual environment tool such as `venv` or `conda`

> **Note:** Some dependencies (PaddleOCR, PaddlePaddle, Torch) are large. Total install size can exceed 3 GB. Ensure you have sufficient disk space and a stable internet connection.

### Installation

**1. Clone the repository**

```bash
git clone https://github.com/your-username/smart-expense-tracker.git
cd "smart-expense-tracker/Smart Expense Tracker With ML"
```

**2. Create and activate a virtual environment**

```bash
python -m venv venv

# On Windows:
venv\Scripts\activate

# On macOS/Linux:
source venv/bin/activate
```

**3. Install dependencies**

```bash
pip install -r requirements.txt
```

**4. Download the spaCy language model**

```bash
python -m spacy download en_core_web_sm
```

**5. Apply database migrations**

```bash
python manage.py migrate
```

**6. (Optional) Create a superuser for admin access**

```bash
python manage.py createsuperuser
```

### Running the Application

```bash
python manage.py runserver
```

Open your browser and navigate to `http://127.0.0.1:8000/`.

The Django admin panel is available at `http://127.0.0.1:8000/admin/`.

## 🤖 Machine Learning Modules

### 1. Spending Forecast (`ml_engine.py` — `FinancialAdvisor.get_spending_forecast`)

Uses a **Random Forest Regressor** (`sklearn.ensemble.RandomForestRegressor`) to predict total spending for the next 7 days.

**Features used:**

| Feature | Description |
|---|---|
| `day_of_week` | Day of the week (0 = Monday, 6 = Sunday) |
| `day_of_month` | Calendar day (1–31) |
| `is_weekend` | Binary flag for Saturday/Sunday |

Requires a minimum of **10 historical expense records** to generate a prediction.

### 2. Savings Recommendations (`ml_engine.py` — `FinancialAdvisor.get_savings_recommendations`)

Generates up to 3 actionable tips by:
- Comparing spending per category over the current vs. previous 30 days
- Flagging categories with a >20% increase (Medium) or >50% increase (High)
- Issuing a Critical alert when >90% of the monthly budget is consumed
- Suggesting bulk purchasing for the most frequently transacted category

### 3. NLP Expense Parsing (`nlp_utils.py` — `parse_expense_text`)

Processes free-form text input through a three-stage pipeline:

1. **Amount extraction** — Regex-based numeric extraction
2. **Date extraction** — spaCy NER for relative dates ("yesterday", "today") with Hindi keyword fallback ("कल", "आज")
3. **Category classification** — Zero-shot classification via `typeform/distilbert-base-uncased-mnli` across 21 predefined categories, with Hindi-to-English keyword mapping as preprocessing

## ⚙️ Configuration

Key settings are in `expense_tracker/settings.py`:

| Setting | Default | Notes |
|---|---|---|
| `DEBUG` | `True` | Set to `False` in production |
| `SECRET_KEY` | (insecure default) | **Replace with a secure key before deployment** |
| `ALLOWED_HOSTS` | `['*']` | Restrict to your domain in production |
| `TIME_ZONE` | `Asia/Kolkata` | Update for your region |
| `DATABASES` | SQLite | Switch to PostgreSQL for production use |
| `MEDIA_ROOT` | `./media/` | Storage path for uploaded receipt images |

> ⚠️ **Security Notice:** The default `SECRET_KEY` in `settings.py` is insecure and must be replaced before deploying to any public environment. Use environment variables (e.g., via `python-decouple` or `django-environ`) to manage secrets safely.
